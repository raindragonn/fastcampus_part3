## chapter02_오늘의 명언

> FireBase의 Remote Config를 활용해 별도의 코드 수정이나 배포없이 앱의 내용을 수정할수 있다.


**활용 기술**

- `Firebase Remote Config`

    - 앱 업데이트를 하지 않아도 앱의 동작과 모양을 변경할 수 있다.

    - 최대 2000개의 매개변수와 최대 500개의 조건 까지 구성 가능하다.

- `ViewPager2`

    - ViewPager 에서 개선된 버전
  
    - 가로,세로 방향 모두 지원. RTL 지원
  
    - recyclerView 를 기반으로 되어있다.
